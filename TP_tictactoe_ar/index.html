    Exercice sommatif - Tic Tac Toe en Réalité Augmentée

    Contexte et objectifs

    Vous allez créer une application complète de Tic Tac Toe en AR qui combine tous les concepts vus durant les trois dernières semaines. Cette application permettra à deux joueurs de jouer au Tic Tac Toe sur une grille placée sur une surface réelle détectée.

    Objectifs pédagogiques

    • ✅ Maîtriser la détection de plans AR
    • ✅ Implémenter le raycasting AR pour les interactions
    • ✅ Gérer la logique de jeu (états, tours, victoire)
    • ✅ Créer une interface utilisateur claire et adaptée à l'AR
    • ✅ Respecter les principes de design des applications AR
    • ✅ Appliquer les bonnes pratiques de développement Unity

    Spécifications du projet

    Fonctionnalités obligatoires

    1. Détection et placement de la grille

    • [ ] Détection des plans horizontaux (sols, tables)
    • [ ] Visualisation des plans détectés (feedback visuel)
    • [ ] Placement de la grille de jeu sur un plan par tap
    • [ ] La grille reste fixe une fois placée (utilisation d'ancres)
    • [ ] Option pour repositionner la grille (bouton "Réinitialiser")

    2. Logique de jeu

    • [ ] Alternance entre joueur X et joueur O
    • [ ] Placement de symboles (X ou O) par tap sur une case
    • [ ] Vérification des cases déjà occupées (tap ignoré)
    • [ ] Détection automatique de victoire (3 alignés)
    • [ ] Détection de match nul (grille complète sans gagnant)
    • [ ] Possibilité de recommencer une partie

    3. Interface utilisateur

    • [ ] Indicateur du joueur actuel ("Tour de X" / "Tour de O")
    • [ ] Message de fin de partie ("X a gagné!" / "Match nul!")
    • [ ] Bouton "Nouvelle partie"
    • [ ] Bouton "Repositionner la grille"
    • [ ] Instructions initiales ("Scannez une surface...")

    4. Feedback visuel et sonore

    • [ ] Animation de placement des symboles
    • [ ] Mise en évidence de la ligne gagnante
    • [ ] Son au placement d'un symbole (optionnel)
    • [ ] Son de victoire (optionnel)
    • [ ] Effet visuel sur la case sélectionnée

    TicTacToeBoard
├── BoardBase (Plane ou Quad)
│   └── Matériau : Blanc semi-transparent
├── GridLines (9 cubes fins pour les lignes)
└── CellPositions (9 GameObjects vides)
    ├── Cell_0_0 (Position : -0.15, 0.01, 0.15)
    ├── Cell_0_1 (Position : 0, 0.01, 0.15)
    ├── Cell_0_2 (Position : 0.15, 0.01, 0.15)
    ├── Cell_1_0 (Position : -0.15, 0.01, 0)
    ├── Cell_1_1 (Position : 0, 0.01, 0)
    ├── Cell_1_2 (Position : 0.15, 0.01, 0)
    ├── Cell_2_0 (Position : -0.15, 0.01, -0.15)
    ├── Cell_2_1 (Position : 0, 0.01, -0.15)
    └── Cell_2_2 (Position : 0.15, 0.01, -0.15)

    Dimensions suggérées : - Grille totale : 0.5m × 0.5m - Chaque case : ~0.15m × 0.15m - Hauteur au-dessus de la surface : 0.01m

    Script sur la grille :

    using UnityEngine;

public class TicTacToeBoard : MonoBehaviour
{
    [SerializeField] private Transform[] cellPositions = new Transform[9];

    public Transform GetCellPosition(int index)
    {
        if (index >= 0 && index < cellPositions.Length)
            return cellPositions[index];
        return null;
    }

    public Vector3 GetCellWorldPosition(int index)
    {
        Transform cell = GetCellPosition(index);
        return cell != null ? cell.position : Vector3.zero;
    }
}

    Phase 3 : Placement AR de la grille (60 min)

    Étape 3.1 : Script de placement (ARPlacementManager.cs)

    using UnityEngine;
using UnityEngine.XR.ARFoundation;
using UnityEngine.XR.ARSubsystems;
using System.Collections.Generic;

public class ARPlacementManager : MonoBehaviour
{
    [Header("AR Components")]
    [SerializeField] private ARRaycastManager raycastManager;
    [SerializeField] private ARPlaneManager planeManager;
    [SerializeField] private ARAnchorManager anchorManager;

    [Header("Prefabs")]
    [SerializeField] private GameObject boardPrefab;

    [Header("Settings")]
    [SerializeField] private bool placementEnabled = true;

    private GameObject currentBoard;
    private ARAnchor currentAnchor;
    private List<ARRaycastHit> hits = new List<ARRaycastHit>();

    public bool IsBoardPlaced => currentBoard != null;
    public GameObject CurrentBoard => currentBoard;

    void Update()
    {
        if (!placementEnabled || currentBoard != null)
            return;

        // Vérifier les taps (adapté pour desktop et mobile)
        if (Input.touchCount > 0)
        {
            Touch touch = Input.GetTouch(0);
            if (touch.phase == TouchPhase.Began)
            {
                TryPlaceBoard(touch.position);
            }
        }
        else if (Input.GetMouseButtonDown(0))
        {
            TryPlaceBoard(Input.mousePosition);
        }
    }

    private void TryPlaceBoard(Vector2 screenPosition)
    {
        // Raycast AR vers les plans détectés
        if (raycastManager.Raycast(screenPosition, hits, TrackableType.PlaneWithinPolygon))
        {
            Pose hitPose = hits[0].pose;

            // Créer une ancre pour stabiliser la position
            currentAnchor = anchorManager.AddAnchor(hitPose);

            if (currentAnchor != null)
            {
                // Instancier la grille à la position de l'ancre
                currentBoard = Instantiate(boardPrefab, currentAnchor.transform);
                currentBoard.transform.localPosition = Vector3.zero;
                currentBoard.transform.localRotation = Quaternion.identity;

                // Désactiver la visualisation des plans (optionnel)
                SetPlanesActive(false);

                Debug.Log("Grille placée avec succès!");

                // Notifier le GameController
                GameController gameController = FindObjectOfType<GameController>();
                if (gameController != null)
                {
                    gameController.OnBoardPlaced(currentBoard);
                }
            }
        }
    }

    public void ResetPlacement()
    {
        if (currentBoard != null)
        {
            Destroy(currentBoard);
            currentBoard = null;
        }

        if (currentAnchor != null)
        {
            Destroy(currentAnchor.gameObject);
            currentAnchor = null;
        }

        // Réactiver la visualisation des plans
        SetPlanesActive(true);
        placementEnabled = true;

        Debug.Log("Placement réinitialisé");
    }

    private void SetPlanesActive(bool active)
    {
        foreach (var plane in planeManager.trackables)
        {
            plane.gameObject.SetActive(active);
        }
    }
}

    Phase 4 : Logique du jeu (90 min)

    Étape 4.1 : Script principal (GameController.cs)

    using UnityEngine;
using UnityEngine.XR.ARFoundation;
using System.Collections.Generic;
using UnityEngine.UI;
using TMPro;

public class GameController : MonoBehaviour
{
    [Header("References")]
    [SerializeField] private ARRaycastManager raycastManager;
    [SerializeField] private ARPlacementManager placementManager;

    [Header("Prefabs")]
    [SerializeField] private GameObject xSymbolPrefab;
    [SerializeField] private GameObject oSymbolPrefab;

    [Header("UI")]
    [SerializeField] private TextMeshProUGUI currentPlayerText;
    [SerializeField] private TextMeshProUGUI instructionsText;
    [SerializeField] private GameObject gameOverPanel;
    [SerializeField] private TextMeshProUGUI winnerText;

    private TicTacToeBoard board;
    private GameObject[,] grid = new GameObject[3, 3];
    private int[,] gameState = new int[3, 3]; // 0 = vide, 1 = X, 2 = O
    private bool isXTurn = true;
    private bool gameActive = false;
    private List<ARRaycastHit> hits = new List<ARRaycastHit>();

    void Start()
    {
        UpdateInstructions("Scannez une surface pour placer la grille...");
        gameOverPanel.SetActive(false);
    }

    void Update()
    {
        if (!gameActive || !placementManager.IsBoardPlaced)
            return;

        // Détection des taps pour placer les symboles
        if (Input.touchCount > 0)
        {
            Touch touch = Input.GetTouch(0);
            if (touch.phase == TouchPhase.Began)
            {
                HandleTap(touch.position);
            }
        }
        else if (Input.GetMouseButtonDown(0))
        {
            HandleTap(Input.mousePosition);
        }
    }

    public void OnBoardPlaced(GameObject placedBoard)
    {
        board = placedBoard.GetComponent<TicTacToeBoard>();
        gameActive = true;
        UpdateInstructions("");
        UpdateCurrentPlayerDisplay();
    }

    private void HandleTap(Vector2 screenPosition)
    {
        // Raycast classique pour toucher les cases
        Ray ray = Camera.main.ScreenPointToRay(screenPosition);
        RaycastHit hit;

        if (Physics.Raycast(ray, out hit))
        {
            // Vérifier si on a touché une case
            int cellIndex = GetCellIndex(hit.point);

            if (cellIndex != -1)
            {
                int row = cellIndex / 3;
                int col = cellIndex % 3;

                PlaceSymbol(row, col);
            }
        }
    }

    private int GetCellIndex(Vector3 hitPoint)
    {
        // Trouver la case la plus proche du point touché
        float minDistance = float.MaxValue;
        int closestIndex = -1;

        for (int i = 0; i < 9; i++)
        {
            Vector3 cellPos = board.GetCellWorldPosition(i);
            float distance = Vector3.Distance(hitPoint, cellPos);

            if (distance < minDistance && distance < 0.1f) // Seuil de 10cm
            {
                minDistance = distance;
                closestIndex = i;
            }
        }

        return closestIndex;
    }

    private void PlaceSymbol(int row, int col)
    {
        // Vérifier si la case est vide
        if (gameState[row, col] != 0)
        {
            Debug.Log("Case déjà occupée!");
            return;
        }

        // Créer le symbole
        GameObject symbolPrefab = isXTurn ? xSymbolPrefab : oSymbolPrefab;
        int cellIndex = row * 3 + col;
        Vector3 cellPosition = board.GetCellWorldPosition(cellIndex);

        GameObject symbol = Instantiate(symbolPrefab, cellPosition, Quaternion.identity);
        symbol.transform.parent = board.transform;

        // Mettre à jour l'état du jeu
        grid[row, col] = symbol;
        gameState[row, col] = isXTurn ? 1 : 2;

        // Vérifier victoire ou match nul
        if (CheckWin())
        {
            EndGame(isXTurn ? "X a gagné!" : "O a gagné!");
            return;
        }

        if (CheckDraw())
        {
            EndGame("Match nul!");
            return;
        }

        // Changer de joueur
        isXTurn = !isXTurn;
        UpdateCurrentPlayerDisplay();
    }

    private bool CheckWin()
    {
        int player = isXTurn ? 1 : 2;

        // Vérifier lignes
        for (int row = 0; row < 3; row++)
        {
            if (gameState[row, 0] == player && 
                gameState[row, 1] == player && 
                gameState[row, 2] == player)
                return true;
        }

        // Vérifier colonnes
        for (int col = 0; col < 3; col++)
        {
            if (gameState[0, col] == player && 
                gameState[1, col] == player && 
                gameState[2, col] == player)
                return true;
        }

        // Vérifier diagonales
        if (gameState[0, 0] == player && 
            gameState[1, 1] == player && 
            gameState[2, 2] == player)
            return true;

        if (gameState[0, 2] == player && 
            gameState[1, 1] == player && 
            gameState[2, 0] == player)
            return true;

        return false;
    }

    private bool CheckDraw()
    {
        for (int row = 0; row < 3; row++)
        {
            for (int col = 0; col < 3; col++)
            {
                if (gameState[row, col] == 0)
                    return false;
            }
        }
        return true;
    }

    private void EndGame(string message)
    {
        gameActive = false;
        winnerText.text = message;
        gameOverPanel.SetActive(true);
    }

    public void NewGame()
    {
        // Nettoyer les symboles existants
        for (int row = 0; row < 3; row++)
        {
            for (int col = 0; col < 3; col++)
            {
                if (grid[row, col] != null)
                {
                    Destroy(grid[row, col]);
                    grid[row, col] = null;
                }
                gameState[row, col] = 0;
            }
        }

        isXTurn = true;
        gameActive = true;
        gameOverPanel.SetActive(false);
        UpdateCurrentPlayerDisplay();
    }

    public void ResetPlacement()
    {
        NewGame();
        placementManager.ResetPlacement();
        UpdateInstructions("Scannez une surface pour placer la grille...");
    }

    private void UpdateCurrentPlayerDisplay()
    {
        currentPlayerText.text = isXTurn ? "Tour de X" : "Tour de O";
        currentPlayerText.color = isXTurn ? Color.blue : Color.red;
    }

    private void UpdateInstructions(string text)
    {
        instructionsText.text = text;
    }
}

    Aide au débogage

    Simulation ne fonctionne pas : 1. Vérifiez XR Plug-in Management → Desktop → XR Simulation ✓ 2. Vérifiez qu'un environnement est sélectionné

    Raycast ne détecte rien : 1. Vérifiez que AR Raycast Manager est sur XR Origin 2. Vérifiez que les plans sont détectés (grilles visibles) 3. Console : regardez les erreurs

    Symboles n'apparaissent pas : 1. Vérifiez que les prefabs sont assignés 2. Vérifiez que le raycast touche bien une case (Debug.Log) 3. Vérifiez que la caméra voit les symboles (position Y)

    Conseils de développement

    Testez progressivement

    Ne construisez pas tout d'un coup. Testez après chaque phase : 1. Placement de grille → Test 2. Ajout d'un symbole → Test 3. Logique de victoire → Test 4. UI → Test

    Utilisez la simulation

    Développez 100% en simulation

    Commits réguliers

    Faites des commits GitHub à chaque étape fonctionnelle : - "Détection de plans fonctionnelle" - "Placement de grille implémenté" - "Logique de jeu complétée"

    Demandez de l'aide tôt

    Ne restez pas bloqué 2 heures sur un problème. Demandez de l'aide après 30 minutes.
